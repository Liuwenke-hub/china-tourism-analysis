# 中国旅游经济分析 · 交互式数据仪表盘

> **基于 Python（pandas + pyecharts + matplotlib）的多维旅游经济可视化分析项目**
> 数据覆盖 2005–2022 年，涵盖宏观经济、国内/国际旅游、区域差异与海南专题。

## 项目背景

本项目是我在大学期间（2021–2023，旅游管理专业）完成的「大数据可视化课程设计」。从一个单文件 matplotlib 静态图表脚本出发，逐步迭代为五层模块化架构，并新增了 pyecharts 交互式仪表盘与 python-docx 自动报告生成功能。数据窗口统一截断至 2022 年（本人 2023 年 6 月毕业前可获得的公开数据）。

---

### 分析动机

作为一名**旅游管理专业背景**的数据分析学习者，我注意到国内关于"旅游经济与宏观发展关系"的分析多停留在定性描述或单一指标层面，缺乏一个**跨维度、可交互、可复现**的量化分析工具。本项目从公开统计年鉴中整合了 **10 类数据集**，通过 ETL 清洗、相关性建模与多类型可视化，系统揭示：

- 居民收入增长如何传导至旅游消费升级；
- 疫情对国内旅游的冲击（2020–2022 两年持续承压）；
- 区域间旅游外汇收入的马太效应；
- 海南作为国际旅游岛的月度季节性规律。

这些结论为文旅目的地运营、客源结构优化和淡旺季调度提供了**可量化的决策参考**。

## 技术栈

| 层级 | 工具 | 用途 |
|------|------|------|
| 数据处理 | `pandas` + `numpy` + `openpyxl` | Excel 宽表→长表 melt、省份清洗、ETL 解析 |
| 交互可视化 | `pyecharts` v2.x | 13 张可交互 HTML 图表（悬停/缩放/联动） |
| 静态细节图 | `matplotlib` + `seaborn` | 30 张 300 DPI 静态 PNG（1:1 复刻原项目，直接入报告） |
| 报告生成 | `python-docx` | 一键生成《可视化分析报告》docx（含 30 图 + 分析） |
| 语言 | Python 3.10+ | 全模块化，无硬编码 |

## 核心分析模块

### 1. 宏观经济发展趋势
- GDP 总量与增长率双轴折线图（2005–2022，123.4 万亿元峰值）
- 居民人均可支配收入 vs 消费水平趋势（收入弹性分析）

### 2. 旅游业发展趋势
- **国内旅游**：人次与总花费（2020 年疫情断崖 -52%，2022 年再度承压 -22%）
- **国际旅游**：外汇收入与接待国外游客（2005–2019，疫情前基线）

### 3. 旅游与国民经济相关性
- Pearson 相关性热力图（GDP↔旅游消费 r≈0.95+）
- GDP vs 国际旅游外汇收入散点 + 线性回归拟合
- GDP vs 国内旅游人次散点回归

### 4. 区域差异分析
- 各省 GDP 排名 Top15（广东/江苏/山东领跑）
- 各省国际旅游外汇收入排名与箱线分布（北京/上海显著领先）
- 重点省份 GDP 增长率对比

### 5. 旅游产业供给
- 旅行社数量与星级饭店总数趋势
- 旅游总收入构成（国内 vs 国际，堆叠面积）

### 6. 海南旅游专题（月度）
- 月度游客与收入趋势（2015–2022）
- 季节性特征：冬季（1–3 月）为旺季，夏季相对平缓

## 关键发现（旅游管理视角）

1. **收入→消费的正向传导链**：居民人均可支配收入每增长 1%，国内旅游花费约增长 1.2%（弹性>1），印证"旅游是可选消费中的刚需化"趋势。
2. **疫情冲击与持续承压**：2020 年国内游客同比下降 52%，2021 年短暂反弹（+12.7%）后，2022 年因疫情反复再度下滑 -22%，两年间旅游市场持续承压、尚未恢复至疫前规模。
3. **区域马太效应**：国际旅游外汇收入高度集中在东部沿海（北京/上海/江苏占全国 >40%），西部省份在国际客源竞争中处于劣势。
4. **海南的季节性窗口**：11 月–次年 3 月为旅游高峰期，6–8 月为相对低谷——这与"避寒游"需求完全吻合，建议目的地运营据此调配资源。

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 生成全部成果（交互仪表盘 + 30 张静态图 + 报告）
python src/dashboard_pyecharts.py    # -> output/dashboard.html（13 张交互图）
python src/plots_matplotlib.py       # -> output/charts/01–30_*.png（30 张静态图）
python src/generate_report.py        # -> output/中国旅游与经济发展可视化分析报告.docx

# 3. 打开仪表盘 / 报告
start output/dashboard.html          # Windows
start output/中国旅游与经济发展可视化分析报告.docx
```
