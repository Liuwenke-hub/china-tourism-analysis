## 摘要 | Abstract

基于国家统计局、文化和旅游部等公开的 **10 类统计年鉴**（2005–2022），构建覆盖宏观经济、国内旅游、国际旅游、区域差异与海南专题的多维旅游经济数据分析体系。采用五层模块化架构（config / data_loader / analysis / dashboard / plots），通过 **pyecharts** 交互式仪表盘与 **matplotlib** 静态图双轨输出，最终以 **python-docx** 自动生成结构化分析报告。

> Based on 10 public statistical datasets (2005–2022) from National Bureau of Statistics and Ministry of Culture and Tourism of PRC, this project builds a multi-dimensional tourism economic analysis system covering macro economy, domestic/international tourism, regional differences and Hainan special topic. It adopts a five-layer modular architecture with dual-track output via pyecharts interactive dashboard and matplotlib static charts, finally generating structured reports automatically through python-docx.

## 背景与动机

从行业视角观察到，国内关于"旅游经济与宏观发展关系"的分析多停留在定性描述或单一指标层面，缺乏一个**跨维度、可交互、可复现**的量化分析工具。

本项目试图回答以下问题：

1. 宏观经济增长如何传导至居民旅游消费升级？
2. 疫情三年（2020–2022）对国内/国际旅游分别造成了多大冲击？恢复情况如何？
3. 旅游外汇收入在省际间呈现怎样的分布特征？是否存在"马太效应"？
4. 海南作为国际旅游岛，其月度游客与收入呈现怎样的季节性规律？

## 关键发现总结

### 发现一：收入→消费的正向传导链

居民人均可支配收入每增长 **1%**，国内旅游总花费约增长 **1.2%**（弹性 > 1），印证"旅游是可选消费中的刚需化"长期趋势。GDP 与旅游消费的 Pearson 相关系数高达 **0.992**（2005–2019），两者几乎同步波动。

### 发现二：疫情冲击与持续承压

| 年份 | 国内游客同比 | 国内旅游花费同比 |
|:----:|:-----------:|:---------------:|
| 2019 | +8.4% | — |
| **2020** | **-52.1%** | **~-50%** |
| 2021 | +12.7%（低基数反弹） | — |
| **2022** | **-22.1%** | **~-20%** |

疫情造成两年断崖式下跌后，2022 年再度承压，表明旅游市场恢复具有**滞后性和脆弱性**。

### 发现三：区域马太效应显著

国际旅游外汇收入的省际分布高度不均：

- **头部集中**：Top3 省份（北京/上海/江苏）占全国总额 **40%+**
- **尾部弱势**：西部多数省份占比 < 1%
- **箱线图揭示**：省际差异达 **20 倍量级**

这一格局对文旅政策制定有直接参考意义——资源应向有潜力的中部省份倾斜，而非持续加码东部。

### 发现四：海南的季节性窗口效应

海南月度数据显示清晰的"冬暖夏凉"模式：

- 旺季（11–3 月）：月均游客 **450 万+人次**，收入峰值明显
- 淡季（6–8 月）：月均游客回落至 **300 万以下**

建议目的地运营据此动态调整定价策略、人员排班和营销投放节奏。

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/Liuwenke-hub/china-tourism-analysis.git
cd china-tourism-analysis

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行全部脚本（按顺序）

# ① 生成交互式仪表盘
python src/dashboard_pyecharts.py
#    -> output/dashboard.html（13 张交互式 HTML 图表）
#    浏览器打开即可查看悬停/缩放/联动效果

# ② 生成静态分析图
python src/plots_matplotlib.py
#    -> output/charts/01_中国GDP总量年度趋势.png ... 30_*.png

# ③ 生成 Word 分析报告
python src/generate_report.py
#    -> output/中国旅游与经济发展可视化分析报告.docx
#    （含 30 张图表 + 6 章文字分析）

# ④ SQL 查询演示（可选）
python src/sql_demo.py
#    控制台输出 4 组 SQL 查询结果

# ⑤ 一键运行全部（推荐）
python run_all.py
```
